# /graphify

Run `graphify query "<question>"` for codebase questions. For rebuilds: `graphify .` or `graphify . --update`.

## Install

```bash
graphify --help || uv tool install graphifyy -q
```

Full docs: https://github.com/safishamsi/graphify
